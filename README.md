## Project Overview

This project analyzes customer experience data to identify the factors that most influence the Net Promoter Score (NPS). The methodology combines exploratory analysis, mathematical optimization and machine learning techniques to support business decision-making.

---

## Objective

Identify the customer experience drivers that have the greatest impact on customer recommendation and satisfaction.

---

## Methodology

1. Survey data preprocessing and cleaning.
2. Exploratory data analysis.
3. Customer segmentation by age groups.
4. Linear programming model in GAMS.
5. Machine learning model training.
6. Results comparison and interpretation.
